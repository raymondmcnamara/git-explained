Setup

Step 1: Install Git

You can download the Windows installer here - https://git-scm.com/downloads/win

Step 2: Add Git to Environment Path

Go the search bar in Windows. Search for "environment variables".

Click on the "Edit the system environment variables" option:

Click on the "Environment Variables" button:

Make sure the Path has an entry for Git:

Now you can run Git in the command line, no matter which directory you're in:

C:\Users\raymc\Pictures>git --version

git version 2.46.2.windows.1

Step 3: Configure Your Credentials

You need to configure your credentials so Git can show your collaborators which commits are yours.

You can set your credentials with the following commands:

git config --global user.name "Your Name"

git config --global user.email "your.email@example.com"
