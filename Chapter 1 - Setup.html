Setup

Step 1: Install Git

You can download the Windows installer here - https://git-scm.com/downloads/win

Step 2: Add Git to Environment Path

Go the search bar in Windows. Search for "environment variables".

Click on the "Edit the system environment variables" option:

Click on the "Environment Variables" button:

Make sure the Path has an entry for Git:

Now you can run Git in the command line, no matter which directory you're in:

C:\Users\raymc\Pictures>git --version

git version 2.46.2.windows.1

Step 3: Configure Your Credentials

You need to configure your credentials so Git can show your collaborators which commits are yours.

You can set your credentials with the following commands:

git config --global user.name "Your Name"

git config --global user.email "your.email@example.com"

Step 5: Push to Remote

Open up settings in your GitHub account:

Scroll down to Developer Settings:

Click on Personal access tokens:

Click on Fine-grained tokens:

Generate a new token:

Enter a name for the new token:

Specify the repository:

Click the button to add repository level permissions:

Add read-only access to metadata:

Add read-write access to commits and contents:

Add read-write access to pull requests:

Generate the token text. The next screen will show you the token text - copy it:

Open Windows Credential Manager and go to the Windows Credentials tab:

Click on "Add a generic credential":

Save a credential like so(use the token you copied as the password):

Go to windows explorer and create a folder with the same name as your repo(in this case, "example-repo"):

Open up a command prompt and go to the folder you just created. Run the command git init to initialise a git repository.

Create a new text file in your folder:

Now run the command git add . to add the new file to the git index.

Run the command git commit -m "first commit" to create a commit.

Run the command git branch -M main to rename the default branch from master to main so that it works with GitHub.

Run the command git remote add origin https://github.com/raymondmcnamara/example-repo.git so that your commit can be pushed to your remote GitHub repository.

Run the command git push -u origin main to push your commit to your remote GitHub repository.
